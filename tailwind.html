Tailwind

Tailwind är ett CSS-ramverk som skiljer sig från andra CSS-ramverk eftersom de inte tillhandahåller en serie förfinade klasser för element, knappar, tabeller. Det finns istället en "verktygslåda" med CSS-klasser för att style varje element själv. Det gör att Tailwind CSS kan anpassas mer. Det kan användas i olika projekt genom att använda CDN, installera via pakethanterare eller ladda ner källkoden.


Tailwindcss officiella hemsida

Fördelen med Tailwindcss är att det går att anpassa designen nästan hur man vill utifrån den "verktygslåda" som finns. Det går snabbare att använda Tailwind än att skriva all CSS-kod själv. Det blir inte lika tydligt att man använt just ramverket Tailwind till sin webbplats. Det finns bra dokumentation.

Nackdelen är att det krävs mer jobb för att få en enklare webbplats och man behöver tänka mer på responsivitet osv. Du får helt enkelt inte lika mycket "gratis" med Tailwind utan du behöver själv lägga ner lite mer tid jämfört med Bootstrap eller Milligram exempelvis. Men då går inte de att anpassa lika mycket istället. Det beror helt enkelt på vad man ska använda ramverket till.
