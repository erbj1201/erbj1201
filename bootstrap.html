Bootstrap

Bootstrap är ett ramverk för CSS. Det kan användas i olika projekt genom att använda CDN, installera via pakethanterare eller ladda ner källkoden. Du kan anpassa Bootstrap med hjälp av Sass och importera de komponter du behöver. Läs mer på Boostraps officiella hemsida.

Jag tycker att ramverket har fördelar så som att det går snabbt att skapa en grund för en enklare webbplats. Dokumentationen är väldigt bra och enkel att följa, lätt att komma igång. Man sparar mycket tid på att inte behöva skriva css för responsivitet, formulär och tabeller. Jag kommer garanterat använda bootstrap igen, eventuellt att jag kan använda Bootstrap till exempelvis formulär och komplettera med egen css.

Några nackdelar är att eftersom jag är van vid att koda allt själv tycker jag att det är lite "låst" jämfört med att skapa något själv. Det går inte att göra "vad som helst". Kanske att det går att anpassa mer om man laddar ner npm-paketet istället för att använda CDN som jag gjorde nu för då kan man skriva egen Sass som transpileras till css. Jag kan komma på flera webbplatser som jag brukar besöka ofta som använder sig av bootstrap. Webbplatserna blir lik varandra vilket är en nackdel. Det är viktigt att skapa en webbplats som sticker ut tycker jag. Det blir också ganska mycket HTML-kod när alla klasser ska vara med för stylingen, mer än när jag gör en design själv.
